# Simulative_ML
## Machine Learning Engineer Course
В этом репозитории будут находится решения Кейсов и Проектов при прохождении курса-симулятора "Инженер машинного обучения" (Simulative, simulative.ru, 2026).

Курс разден на разделы: 
1. Базовый Python + ООП
2. Python для анализа данных
3. Обработка данных для ML
4. Математика для ML
5. Статистика и элементы теории вероятностей
6. Методы оптимизации
7. Обучение с учителем
8. Обучение без учителя
9 Рекомендательные системы (RecSys)
10. Нейронные сети
11. Финальная задача

Каждый раздел содержит главы в которых предлагается решить задачи, кейсы. В репозитории Simulative_ML будут выкладываться кейсы с именами:
S_ML_L_M_N Title,
где S_ML - Simulative_Machine learning, L - номер раздела, M - номер главы, N - номер кейса, Title - заголовок кейса. Для каждого кейса будет отдельный ReadMy с описанием.
